# 关于颜色

由于存在暗色主题，因此需要确保在暗色主题下也能良好显示。

## 颜色变量及其作用

### 可由主题指定的颜色

| 变量名 | 作用 |
| ---- | ---- |
| primary | 主色调 |
| display | 用于显示信息的颜色。例如文字颜色。 |
| display-on-primary | 用于在主色调上显示信息的颜色。 |
| display-hyperlink | 用于显示链接的颜色。 |
| background | 背景色 |
| surface | 比背景色略微凸显的颜色。 |
| warning | 警告色 |
| text-splitter-hover | 分割文本阅读部分的颜色。 |
| active-point-focus | 选中重音短语的颜色。 |
| active-point-hover | 鼠标悬停时重音短语的颜色。 |

### 根据模式变化的颜色

工具栏和分割线的颜色在亮色模式下是主色调，在暗色主题下是表面色。

| 变量名 | 作用 |
| ---- | ---- |
| toolbar | 工具栏颜色 |
| toolbar-button | 工具栏上按钮的颜色 |
| toolbar-button-display | 用于在工具栏按钮上显示信息的颜色 |
| splitter | 分割线颜色 |
